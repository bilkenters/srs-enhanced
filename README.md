## 🇹🇷 Türkçe 

**Bilkent SRS Enhanced**, Bilkent Üniversitesi SRS (Student Review System) arayüzünü modernize eden ve öğrencilerin hayatını kolaylaştıran açık kaynak kodlu bir tarayıcı eklentisidir. `bilkenters.com` tarafından geliştirilmiştir.

### ✨ Özellikler

- **📅 Yaklaşan Sınavlar Paneli:** Ana sayfada, vize ve finallerin kalan süreleriyle birlikte listelendiği şık bir gösterge paneli (Dashboard) ekler.
- **🏷️ Not ve Devamsızlık Rozetleri (Badges):** Derslerin orijinal arayüzünde AJAX ile yüklenmesini beklemek yerine, harf notlarınızı ve güncel devamsızlık oranlarınızı doğrudan ders isminin yanında konumlandırır.
- **⚙️ Native Aç/Kapat Kontrolü:** Eklenti kullanımını SRS ekranı içerisinden tek tuşla geçici olarak durdurabilir veya başlatabilirsiniz.
- **🛡️ %100 Gizlilik:** Eklenti sadece yerel tarayıcınızda çalışır. Notlarınız, öğrenci numaranız veya şifreniz **asla** başka bir sunucuya gönderilmez.

### 🚀 Geliştirici Kurulumu (Local Installation)

Projeyi kendi bilgisayarınızda derlemek veya test etmek isterseniz:
1. Bu depoyu indirin (`Code -> Download ZIP` veya `git clone`).
2. Chrome tarayıcısında `chrome://extensions/` sayfasına gidin.
3. Sağ üst köşeden **"Geliştirici modu"nu (Developer mode)** açın.
4. **"Paketlenmemiş öğe yükle" (Load unpacked)** butonuna tıklayın.
5. İndirdiğiniz klasörü (`bilkent-srs-enhanced`) seçin. Hepsi bu kadar!

## 🇬🇧 English

**Bilkent SRS Enhanced** is an open-source browser extension that modernizes the Bilkent University SRS (Student Review System) interface, making it more robust and productive for students. Proudly developed by `bilkenters.com`.

### ✨ Features

- **📅 Upcoming Exams Dashboard:** Injects a sleek dashboard right onto the main layout displaying midterms and finals along with countdowns.
- **🏷️ Grade & Attendance Badges:** Fetches internal grades and attendance rates dynamically and displays them as inline badges next to the course names.
- **⚙️ Native Toggle:** Temporarily disable/enable the extension natively within the SRS UI with a single click.
- **🛡️ 100% Privacy:** The extension operates purely strictly on the client-side. Your grades, student ID, and passwords are **never** harvested or sent to external servers.

### 🚀 Developer Installation

If you want to run from source or contribute:
1. Clone or download this repository.
2. Go to `chrome://extensions/` in your Chrome browser.
3. Enable **"Developer mode"** in the top right.
4. Click **"Load unpacked"**.
5. Select the `bilkent-srs-enhanced` folder. You are done!
